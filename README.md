# eBay Automation Testing Framework

A comprehensive Selenium WebDriver automation testing suite for eBay.com functionality, built with Java, TestNG, and Maven.


## 📋 Project Overview

This project implements automated end-to-end testing for eBay's core features including search, filters, cart management, product pages, user authentication, watchlist, and account settings.


## 🛠️ Technologies & Tools

- **Java** (JDK 25.0.2)
- **Selenium WebDriver** (v4.41.0)
- **TestNG** - Testing framework
- **Maven** - Build and dependency management
- **ChromeDriver** (v146)
- **IntelliJ IDEA** - IDE
- **Git** - Version control
- **Windows 11** - OS


## 🚀 Getting Started

### Prerequisites
- Java JDK 25 or higher
- Maven 3.6+
- Chrome Browser (v146+)
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/ebay-automation.git
cd ebay-automation
```

2. Update credentials in test files
```java
// Update in LoginEbayTest.java and WatchlistEbayTest.java
private static final String VALID_EMAIL = "your-email@example.com";
private static final String VALID_PASSWORD = "your-password";
```
